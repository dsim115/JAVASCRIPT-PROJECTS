# JAVASCRIPT-PROJECTS

## Introduction
This repository contains the projects I completed during my JavaScript and web development coursework. These projects demonstrate my understanding of front-end development concepts, responsive design, and client-side interactivity using JavaScript and modern web technologies.



## Projects


### Academy Cinemas Website
A responsive movie theater website built using Bootstrap and JavaScript to simulate a real cinema experience.

Features:
- Responsive layout using Bootstrap 5
- Navigation bar with dropdown menus
- Movie listings with showtimes
- Ticket selection interface (movie, time, quantity)
- JavaScript-powered popovers and toast notifications
- Contact form layout

Technologies:
- HTML
- CSS
- Bootstrap 5
- JavaScript
- jQuery



### Pizza Menu
A dynamic pizza ordering interface that allows users to customize their order and calculates the total cost based on selected options.

Features:
- Multiple topping selections
- Price calculation logic
- Interactive user interface
- JavaScript-driven order processing

Technologies:
- HTML
- CSS
- JavaScript



### Tic-Tac-Toe Game
A browser-based version of the classic Tic-Tac-Toe game with added interactivity.

Features:
- Randomized player selection
- Player token selection
- Win detection logic
- Visual and audio feedback for game outcomes

Technologies:
- HTML
- CSS
- JavaScript



### Calculator
A simple web-based calculator that performs basic arithmetic operations.

Features:
- Addition, subtraction, multiplication, and division
- Clean, user-friendly interface
- JavaScript logic for calculations

Technologies:
- HTML
- CSS
- JavaScript
